Backend Developer

In my role as a backend developer for the AmongTrade project, I was responsible for developing the API for a mobile application that focuses on binary options trading. AmongTrade is designed to provide users with a seamless and intuitive platform for engaging in binary options, offering real-time trading capabilities and comprehensive market insights. This involved using Node.js with TypeScript as the primary programming language, ensuring robust and scalable backend functionality. This app chose PostgreSQL as the database management system to handle the application's data efficiently and securely.

To maintain a high standard of code quality and organization, This app implemented the clean architecture approach. This architectural pattern allowed for clear separation of concerns, making the codebase more maintainable and easier to extend. Additionally, This app employed the factory design pattern to streamline the creation of objects, ensuring consistency and reducing the potential for errors throughout the development process.

AmongTrade is designed to offer a seamless and reliable user experience, and my contributions to the backend infrastructure were crucial in achieving this goal. By leveraging advanced design principles and utilizing a powerful tech stack, I was able to develop an API that not only meets the application's requirements but also provides a solid foundation for future enhancements and scalability.
