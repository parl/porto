What's this

Akupon is a third-party application that provides various information and vouchers for shopping. This project represents my final project in the UI/UX Design & Research bootcamp organized by Binar Academy.

The collaboration among UX Research, UX Design, UI Design, and the Product Manager resulted in an exceptional project named Akupon. Cooperation and responsibility were the top priorities within this team.

This project successfully became one of the best final projects in Binar Academy Batch 2, and it is a source of pride to be a part of it.

Role & Design

Design is my life, I take on the role of UI Designer in creating mobile and landing page designs, while also crafting the brand guidelines for Akupon. The first stage involves creating a logo, determining colors, font types to be used, and more.

Moving on to the mobile UI design phase, I start with informational sketches, wireframes, wireflows, high-fidelity designs, and prototyping. Subsequently, the landing page is created based on the values of the Akupon application, with one of its slogans being "Your Wallet's Best Friend."
